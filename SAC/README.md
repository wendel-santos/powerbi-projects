# Relatório SAC — Power BI

Dashboard de análise de atendimentos ao usuário 
desenvolvido no Power BI.

## Indicadores analisados
- Total de chamados respondidos
- Média de chamados e respostas diárias
- Tempo médio de atendimento por mês
- Chamados por tipo de problema
- Performance por atendente

## Ferramentas
Power BI · DAX
